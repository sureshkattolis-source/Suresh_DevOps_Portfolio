🚀 FASTEST WAY TO GET STARTED (Choose 1):
================================================================================

OPTION A: Windows Users
   1. Double-click: start-server.bat
   2. Open browser: http://localhost:8000
   3. Done! ✓

OPTION B: Mac/Linux Users
   1. In terminal, run:
      chmod +x start-server.sh
      ./start-server.sh
   2. Open browser: http://localhost:8000
   3. Done! ✓

OPTION C: Any OS (Python)
   1. Open terminal/command prompt
   2. Navigate to portfolio folder
   3. Run: python -m http.server 8000
      (or: python3 -m http.server 8000 on Mac)
   4. Open browser: http://localhost:8000
   5. Done! ✓

OPTION D: VS Code
   1. Install "Live Server" extension
   2. Right-click index.html
   3. Click "Open with Live Server"
   4. Done! ✓

🔍 TROUBLESHOOTING:
================================================================================

❌ CSS not showing (looks plain)?
   → Make sure you're using local server (not file://)
   → Refresh browser cache (Ctrl+Shift+Delete)

❌ Mobile menu not appearing?
   → Test on mobile or DevTools (F12)
   → Works at 768px and below

❌ Form not working?
   → Check browser console (F12)
   → Currently logs to console

❌ Colors look different?
   → Clear cache and hard refresh (Ctrl+Shift+R)
   → Try different browser

See README.md for detailed troubleshooting.
